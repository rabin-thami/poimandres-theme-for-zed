# Poimandres Theme for Zed

A port of the popular [poimandres](https://github.com/drcmda/poimandres-theme) VSCode theme by [drcmda](https://github.com/drcmda) for the [Zed](https://zed.dev) editor.

> Minimal, frameless dark theme that focuses on semantic meaning instead of color variety.

## Color Semantics

- **Errors / voids / throws / deletes** → red (`#d0679d`)
- **Functions / constructors** → mint (`#5DE4c7`)
- **Keywords** → light blue (`#ADD7FF`)
- **Types** → darker gray (`#767c9d`) — spotlight stays on the code
- **Strings** → muted teal (`#5fb3a1`)
- **Numbers** → pink (`#f087bd`)
- **Comments** → dim blue-gray, italic (`#506477`)
- **Operators / brackets** → sky blue (`#89ddff`)

## Installation

Search for **Poimandres** in the Zed Extensions store (`zed: extensions`).

Or install manually — copy `themes/poimandres.json` to `~/.config/zed/themes/`.
